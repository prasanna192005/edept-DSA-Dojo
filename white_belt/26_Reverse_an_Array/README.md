# Problem 26: Reverse an Array (Edept White Belt)

## Problem Statement
Reverse the elements of a given array.

### Input Format
First line: `N`. Second line: `N` space-separated integers.

### Output Format
The `N` integers of the reversed array, space-separated.

### Example

**Input**
5
1 2 3 4 5

**Output**
5 4 3 2 1


### Constraints
- 1 <= N <= 1000
