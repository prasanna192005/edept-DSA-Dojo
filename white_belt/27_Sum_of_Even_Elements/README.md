# Problem 27: Sum of Even Elements (Edept White Belt)

## Problem Statement
Calculate the sum of only the even numbers in an array.

### Input Format
First line: `N`. Second line: `N` space-separated integers.

### Output Format
The sum of even elements.

### Example

**Input**
6
1 2 3 4 5 6

**Output**
12


### Constraints
- 1 <= N <= 1000
