# Problem 09: Temperature Conversion (Edept White Belt)

## Problem Statement
Convert a given temperature from Celsius to Fahrenheit using the formula: `F = C * 9/5 + 32`.

### Input Format
A single number `C` (Celsius).

### Output Format
The temperature in Fahrenheit.

### Example

**Input**
37

**Output**
98.6


### Constraints
- -100.0 <= C <= 100.0
