# Problem 44: Print All Divisors (Edept White Belt)

## Problem Statement
Given a number `N`, print all of its divisors in ascending order.

### Input Format
A single integer `N`.

### Output Format
The divisors of `N`, space-separated.

### Example

**Input**
100

**Output**
1 2 4 5 10 20 25 50 100


### Constraints
- 1 <= N <= 10000
