# Problem 19: Print Even Numbers (Edept White Belt)

## Problem Statement
Print all even numbers from 1 to `N`.

### Input Format
A single integer `N`.

### Output Format
All even numbers up to `N`, each on a new line.

### Example

**Input**
10

**Output**
2
4
6
8
10


### Constraints
- 1 <= N <= 1000
