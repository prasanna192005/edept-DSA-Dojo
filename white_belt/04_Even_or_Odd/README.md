# Problem 04: Even or Odd (Edept White Belt)

## Problem Statement
Given an integer `N`, determine if it is "Even" or "Odd".

### Input Format
A single integer `N`.

### Output Format
Print "Even" or "Odd".

### Example

**Input**
7

**Output**
Odd


### Constraints
- -1,000,000,000 <= N <= 1,000,000,000
