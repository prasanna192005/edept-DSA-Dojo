# Problem 40: Print 1 to N Recursive (Edept White Belt)

## Problem Statement
Print numbers from 1 to `N` using a recursive function.

### Input Format
A single integer `N`.

### Output Format
Numbers from 1 to `N`, each on a new line.

### Example

**Input**
5

**Output**
1
2
3
4
5


### Constraints
- 1 <= N <= 1000
