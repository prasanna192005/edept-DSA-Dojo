# Problem 42: GCD Euclidean Algorithm (Edept White Belt)

## Problem Statement
Find the Greatest Common Divisor (GCD) of two numbers.

### Input Format
Two space-separated integers `a` and `b`.

### Output Format
The GCD of `a` and `b`.

### Example

**Input**
54 24

**Output**
6


### Constraints
- 1 <= a, b <= 1,000,000,000
