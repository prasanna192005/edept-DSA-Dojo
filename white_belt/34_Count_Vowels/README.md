# Problem 34: Count Vowels (Edept White Belt)

## Problem Statement
Count the number of vowels (a, e, i, o, u) in a string. The check should be case-insensitive.

### Input Format
A single string `S`.

### Output Format
The count of vowels.

### Example

**Input**
Programming

**Output**
3


### Constraints
- String length is between 1 and 1000.
