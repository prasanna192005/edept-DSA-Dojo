# Problem 13: Multiplication Table (Edept White Belt)

## Problem Statement
Given an integer `N`, print its multiplication table from 1 to 10.

### Input Format
A single integer `N`.

### Output Format
The table in the format 'N x i = result'.

### Example

**Input**
5

**Output**
5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50


### Constraints
- 1 <= N <= 1000
