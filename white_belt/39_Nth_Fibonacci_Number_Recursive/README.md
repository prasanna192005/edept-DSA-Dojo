# Problem 39: Nth Fibonacci Number Recursive (Edept White Belt)

## Problem Statement
Find the Nth Fibonacci number using recursion. The sequence starts F(0)=0, F(1)=1.

### Input Format
A single integer `N`.

### Output Format
The Nth Fibonacci number.

### Example

**Input**
7

**Output**
13


### Constraints
- 0 <= N <= 30
