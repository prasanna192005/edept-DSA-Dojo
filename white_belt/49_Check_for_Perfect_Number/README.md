# Problem 49: Check for Perfect Number (Edept White Belt)

## Problem Statement
Check if a number is a Perfect Number (a positive integer equal to the sum of its proper positive divisors).

### Input Format
A single integer `N`.

### Output Format
Print "Yes" or "No".

### Example

**Input**
6

**Output**
Yes


### Constraints
- 1 <= N <= 100000
