# Problem 28: Average of Array (Edept White Belt)

## Problem Statement
Calculate the average of all elements in an array.

### Input Format
First line: `N`. Second line: `N` space-separated integers.

### Output Format
The average of the elements (can be a decimal).

### Example

**Input**
5
1 2 3 4 6

**Output**
3.2


### Constraints
- 1 <= N <= 1000
