# Problem 11: Print Numbers 1 to N (Edept White Belt)

## Problem Statement
Given a positive integer `N`, print all numbers from 1 to `N`, each on a new line.

### Input Format
A single integer `N`.

### Output Format
The numbers from 1 to `N`.

### Example

**Input**
5

**Output**
1
2
3
4
5


### Constraints
- 1 <= N <= 1000
