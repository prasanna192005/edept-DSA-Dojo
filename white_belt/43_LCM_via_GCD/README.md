# Problem 43: LCM via GCD (Edept White Belt)

## Problem Statement
Find the Least Common Multiple (LCM) of two numbers.

### Input Format
Two space-separated integers `a` and `b`.

### Output Format
The LCM of `a` and `b`.

### Example

**Input**
21 6

**Output**
42


### Constraints
- 1 <= a, b <= 100000
