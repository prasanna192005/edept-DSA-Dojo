# Problem 25: Count Occurrences (Edept White Belt)

## Problem Statement
Given an array and a target `X`, count how many times `X` appears.

### Input Format
First line: `N` and `X`. Second line: `N` space-separated integers.

### Output Format
The number of times `X` appears.

### Example

**Input**
6 10
10 20 10 30 10 50

**Output**
3


### Constraints
- 1 <= N <= 1000
