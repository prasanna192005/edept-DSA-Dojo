# Problem 05: Positive Negative or Zero (Edept White Belt)

## Problem Statement
Given an integer `N`, determine if it is "Positive", "Negative", or "Zero".

### Input Format
A single integer `N`.

### Output Format
Print "Positive", "Negative", or "Zero".

### Example

**Input**
-5

**Output**
Negative


### Constraints
- -1,000,000,000 <= N <= 1,000,000,000
