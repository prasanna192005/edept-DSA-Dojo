# Solutions for Problem 1: Hello World

## C Solution
```c
#include <stdio.h>

int main() {
    printf("Hello, Dojo!\n");
    return 0;
}
```

## C++ Solution
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, Dojo!" << std::endl;
    return 0;
}
```

## Java Solution
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Dojo!");
    }
}
```

## Python Solution
```python
print("Hello, Dojo!")
```

## JavaScript Solution
```javascript
console.log("Hello, Dojo!");
```
