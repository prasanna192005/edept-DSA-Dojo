# Problem 44: Sum of Squares

## Problem Statement
Calculate the sum of squares of the first `N` natural numbers.

## Input
A single integer `N`.

## Output
The sum of squares from 1 to `N`.

## Example
**Input:**
```
4
```

**Output:**
```
30
```

## Constraints
1 <= N <= 1000

## Explanation
Use the formula `sum = N * (N + 1) * (2N + 1) / 6`, or compute each square and sum.
