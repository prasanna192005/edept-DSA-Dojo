# Problem 47: Factorial Recursive

## Problem Statement
Calculate the factorial of a number `N` using recursion.

## Input
A single integer `N`.

## Output
The factorial of `N`.

## Example
**Input:**
```
5
```

**Output:**
```
120
```

## Constraints
0 <= N <= 20

## Explanation
Use a recursive function where factorial(N) = N * factorial(N-1), with base case factorial(0) = 1.
