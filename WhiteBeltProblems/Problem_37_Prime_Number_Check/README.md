# Problem 37: Prime Number Check

## Problem Statement
Determine if a given number is prime.

## Input
A single integer `N`.

## Output
Print "Yes" if `N` is prime, "No" otherwise.

## Example
**Input:**
```
17
```

**Output:**
```
Yes
```

## Constraints
1 <= N <= 10^9
