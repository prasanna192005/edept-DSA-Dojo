# Problem 16: Reverse Number

## Problem Statement
Given an integer `N`, reverse its digits and print the result.

## Input
A single integer `N`.

## Output
The reversed integer.

## Example
**Input:**
```
123
```

**Output:**
```
321
```

## Constraints
-1,000,000,000 <= N <= 1,000,000,000

## Explanation
Convert the number to a string and reverse it, or use a mathematical approach by extracting digits with modulo and division.
