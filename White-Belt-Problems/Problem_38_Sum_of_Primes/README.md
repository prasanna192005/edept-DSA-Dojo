# Problem 38: Sum of Primes

## Problem Statement
Find the sum of all prime numbers up to `N`.

## Input
A single integer `N`.

## Output
The sum of all prime numbers from 2 to `N`.

## Example
**Input:**
```
10
```

**Output:**
```
17
```

## Constraints
2 <= N <= 1000

## Explanation
Identify all prime numbers up to N using a primality test, then sum them.
